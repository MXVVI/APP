# Cadastro de Mensagens

Aplicação web simples para cadastro, listagem e exclusão de mensagens.

## Funcionalidades

- Criar mensagem
- Listar mensagens
- Excluir mensagem
- Salvar dados no Supabase

## Tecnologias utilizadas

- HTML
- JavaScript
- Supabase
- Vercel
- GitHub

## Como rodar localmente

```bash
npm install
npm run dev
